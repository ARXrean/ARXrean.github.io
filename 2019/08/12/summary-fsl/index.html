A Survey on Few-Shot Learning

Posted on 2019-08-12

This is a survey of few shot learning problem. Following the route of Wang’s paper and its structure, I read related literatures in it. I will talk some of them in the below.

Data

Augmenting data to a larger size is a very efficient way. Besides the regular geometric transformation, we can use models to learn transformation from original dataset or from other datasets which have similar (weak) labels.

From same dataset

delta-encoder

This paper proposes a variant of an Auto-Encoder network which tries to learn different transformation and applies them to generate new samples from new categories. Suppose there is a large dataset $D$ which has $C$ categories and enough training samples. Besides, there are several categories $C^{few}$ which only contain one or few samples. The delta-encoder try to input $X^a$ and $Y^a$, generating $Z$ with much smaller dimensions. The network try to regrad $Z$ as a tanformation instead of a feature containing the whole info of $X^a$. In decoder, $Z$ is again combined with $Y^a$ and tries to recover $X^a$. The pair samples are randomly sampled from the large dataset and finally become a set $set\lbrace Z\rbrace$.

After training, for generating new samples for new categories, $Y^{new}$ is sampled and combined with sampled $Z$ from $set\lbrace Z\rbrace$ and finally becomde a new feature.

Shrinking and Hallucinating Features

The paper has three contributions.

  1. They create a benchmark for FSL. It contains a large dataset $D$ which has $C_{base}$ categories and enough samples. Besides, it also contains $C_{novel}$ categories and each category has only a few examples. All data are accessed from Imagenet. The process for solving the problem is train feature representation in $C_{base}$->augment samples in $C_{novel}$->learn to distinguish $C_{base}\bigcup C_{novel}$.
  2. They propose to create a network to learn the transfromation in different samples. $X_1$->$X_2$:$Y_1$->$?$. They first cluster 100 groups for each category in $C_{base}$. In training phase,
    $$D(feat_a^A,feat_b^A,feat_a^B)->feat_b^B$$
    And in testing phase,
    $$D(feat_a^A,feat_b^A,feat_a^{novel})->feat_{?}^{novel}$$
  3. For better representation in low-shot learning, they propose a new loss function to regulate the training.
    We know in classical training problem, we want to minimize
    $$\mathop{min}\limits_{W,\Phi} L_D(\Phi,W)=\mathop{min}\limits_{W,\Phi}\frac{1}{|D|}\sum_{(x,y)\in D}L_{cls}(W,\Phi (x),y)$$
    Then each value in final feature can be the probability for the target category
    $$p_k(W,x)=\frac{exp(w_k^T)}{\sum_j exp(w_j^T x)}$$
    In this paper, they regard several tiny sets $S \subset D$(could be one batch) as low-shot dataset. Then they want a classifer $V$ to minimize
    $$\mathop{min}\limits_{V} L_S(\Phi,V)=\mathop{min}\limits_{V}\frac{1}{|S|}\sum_{(x,y)\in S}L_{cls}(V,\Phi (x),y)$$
    Therefore, we want $V$ to match $W$, which means we need $W$ to minimize $L_S(\Phi,V)$. So we want $\triangledown_V L_S(\Phi,V)\mid_{v=w}=0$. Thus we want to minimize
    $$\mathop{\hat{L}}S(\Phi,W)$$
    $\triangledown_V L_S(\Phi,V)$ can be denoted as
    $$\triangledown_V L_S(\Phi,V)=[g_1(S,V),…g_K(S,V)]$$
    $$g_k(S,V)=\frac{1}{|S|}\sum    {(x,y)\in S}(p_k(V,\Phi(x))-\delta_{yk})\Phi(x)$$
    where $\delta_{yk}$ is 1 when $y=k$. If $S$ contains only one sample, then
    $$\hat{L}S(\Phi,W)=\alpha (W,\Phi(x),y)\parallel\Phi(x)\parallel^2$$
    If we average over all samples, then
    $$L_D^{SGM}(\Phi,W)=\frac{1}{|D|}\hat{L}_S(\Phi,W)$$
    We add this to the original loss
    $$\mathop{min}\limits    {W,\Phi} L_D(\Phi,W)+\lambda L_D^{SGM}(\Phi,W)$$
