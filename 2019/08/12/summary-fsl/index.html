A Survey on Few-Shot Learning

Posted on 2019-08-12

This is a survey of few shot learning problem. Following the route of Wang’s paper and its structure, I read related literatures in it. I will talk some of them in the below.

Data

Augmenting data to a larger size is a very efficient way. Besides the regular geometric transformation, we can use models to learn transformation from original dataset or from other datasets which have similar (weak) labels.

From Same Dataset

delta-encoder

This paper proposes a variant of an Auto-Encoder network which tries to learn different transformation and applies them to generate new samples from new categories. Suppose there is a large dataset $D$ which has $C$ categories and enough training samples. Besides, there are several categories $C^{few}$ which only contain one or few samples. The delta-encoder try to input $X^a$ and $Y^a$, generating $Z$ with much smaller dimensions. The network try to regrad $Z$ as a tanformation instead of a feature containing the whole info of $X^a$. In decoder, $Z$ is again combined with $Y^a$ and tries to recover $X^a$. The pair samples are randomly sampled from the large dataset and finally become a set $set\lbrace Z\rbrace$.

After training, for generating new samples for new categories, $Y^{new}$ is sampled and combined with sampled $Z$ from $set\lbrace Z\rbrace$ and finally becomde a new feature.

Shrinking and Hallucinating Features

The paper has three contributions.

  1. They create a benchmark for FSL. It contains a large dataset $D$ which has $C_{base}$ categories and enough samples. Besides, it also contains $C_{novel}$ categories and each category has only a few examples. All data are accessed from Imagenet. The process for solving the problem is train feature representation in $C_{base}$->augment samples in $C_{novel}$->learn to distinguish $C_{base}\bigcup C_{novel}$.

  2. They propose to create a network to learn the transfromation in different samples. $X_1$->$X_2$:$Y_1$->$?$. They first cluster 100 groups for each category in $C_{base}$. In training phase,
    $$
    D(feat_a^A,feat_b^A,feat_a^B)->feat_b^B
    $$
    And in testing phase,
    $$
    D(feat_a^A,feat_b^A,feat_a^{novel})->feat_{?}^{novel}
    $$

  1. For better representation in low-shot learning, they propose a new loss function to regulate the training.

    We know in classical training problem, we want to minimize
    $$
    \mathop{min}\limits_{W,\Phi} L_D(\Phi,W)=\mathop{min}\limits_{W,\Phi}\frac{1}{|D|}\sum_{(x,y)\in D}L_{cls}(W,\Phi (x),y)
    $$
    Then each value in final feature can be the probability for the target category
    $$
    p_k(W,x)=\frac{exp(w_k^T)}{\sum_j exp(w_j^T x)}
    $$
    In this paper, they regard several tiny sets $S \subset D$(could be one batch) as low-shot dataset. Then they want a classifer $V$ to minimize
    $$
    \mathop{min}\limits_{V} L_S(\Phi,V)=\mathop{min}\limits_{V}\frac{1}{|S|}\sum_{(x,y)\in S}L_{cls}(V,\Phi (x),y)
    $$
    Therefore, we want $V$ to match $W$, which means we need $W$ to minimize $L_S(\Phi,V)$. So we want $\triangledown_V L_S(\Phi,V)\mid_{v=w}=0$. Thus we want to minimize
    $$
    LHat_S(\Phi,W)=\parallel\triangledown_V L_S(\Phi,V)\mid_{v=w}\parallel^2
    $$
    $\triangledown_V L_S(\Phi,V)$ can be denoted as
    $$
    \triangledown_V L_S(\Phi,V)=[g_1(S,V),…g_K(S,V)]
    $$

    $$
    g_k(S,V)=\frac{1}{|S|}\sum_{(x,y)\in S}(p_k(V,\Phi(x))-\delta_{yk})\Phi(x)
    $$

    where $\delta_{yk}$ is 1 when $y=k$. If $S$ contains only one sample, then
    $$
    LHat_S(\Phi,W)=\alpha (W,\Phi(x),y)\parallel\Phi(x)\parallel^2
    $$
    If we average over all samples, then
    $$
    L_D^{SGM}(\Phi,W)=\frac{1}{|D|}LHat_S(\Phi,W)
    $$
    We add this to the original loss
    $$
    \mathop{min}\limits_{W,\Phi} L_D(\Phi,W)+\lambda L_D^{SGM}(\Phi,W)
    $$

These methods try to generate more samples for each category to draw their joint distribution. However, they assume the transformation is general over all categories. This is not reasonable for some categories. For example, birds on the ground and birds on the tree can generate one transformation, but this transformation is not suitable for cars.

From Other Dataset

There are also some works trying to borrow samples from other datasets which may have samples with same labels.

large-scale diffusion

This paper tries to support label propagation in a semi-supervised way. Based on few annotated samples and a large set of unlabeled images, they want to propagate corresponing labels.
The standard way of label propagation is:

  1. Convert images to features based on trained CNN network.
  2. Build kNN graph representing the $N\times N$ sparse similarity matrix, denoted by $W$.
  3. Build initial label matrix in which gt-labeled images have one-hot encoding while unlabeled ones are all $0$. Then the known labels are diffused by
    $$
    L_{t+1}=WL_t
    $$
    Note that each row in $L$ corresponds to one sample and we can reset gt-labels to one-hot encoding during iterations.

Then the authors make some variations. These aitional operations can be denoted as
$$
L_{t+1}=\eta(WL_t)
$$
$\eta$ can be different as

  1. a function that $l_1$-normalize each row to provide distribution over labels.
  2. a function that $l_1$-normalize each column if we assume unlabled images have a uniform distribution on labels.
  3. a merged way with above(not understand now).
  4. the Markov Clustering denoted as
    $$
    L_{t+1}=\tau_r(\eta(WL_t))
    $$

Fusing Side information

This paper proposes to convert ‘large-example’ classes to ‘few-example’ classes by using a similarity matrix.

Firstly there are four definitions,

$S$ ${x_i,y_i}_{i=1}^N$ with $C$ classes
$R$ ${R^1,R^2,…,R^M}$
$S’$ ${x_i^{‘},y_i^{‘}}_{i=1}^N$ with $C’$ classes
$R’$ ${R^{‘1},R^{‘2},…,R^{‘M}}$

The goal is to learn the non-linear mapping ${X,X’}$ that aligns with ${R,R’}$.

They use HSIC as statistical dependency on the learned embeddings and labels. denoted as
$$
HSIC(S,R)=\frac{1}{(N-1)^2}tr(HK_GHK_R)
$$
where $K_{G_{ij}}=k_g(x_i,x_j)$, $K_{R_{ij}}=k_r(y_i,y_j)$. The former is inner product of two embeddings while the later is the combination of each $R^m$ denoted as
$$
k_r(y_i,y_j)=\sum_{m=1}^M\frac{1}{M}k_{r^m}(y_i,y_j)
$$
And we are trying to maximize HSIC to get the best representation of the relationship in labels. Finally we can incorporate different set as $HSIC({S,S’},{R,R’})$.

Besides, they propose an attention kernel to convert ‘large-examples’ classes to ‘one-shot’ classes by
$$
y=\sum_{i\in S’}a_r(y,y_i^{‘})y_i^{‘}
$$
These new generated samples can be additional training set.

Model

Few-Shot Adversarial DA

  1. There are two datasets, $D_s={(x_i^s,y_i^s)}{i=1}^{N}$ and $D_t={(x_i^t,y_i^t)}{i=1}^{M}$. The goal is to learn a prediction function $f:\chi \rightarrow \gamma$ that perform well on test data from $D_t$.
  2. We can split $f$ to two parts, $g:\chi \rightarrow \Z$ and $h:\Z \rightarrow \gamma$. DA looks for some inference functions so that $D_t$ can use $h_s$.
  3. Traditional ways try to align distributions of features from $D_s$ to $D_t$. So classifiers are not able to tell where a sample is coming from.
  4. If we use adversarial training, we first train a discriminator $D$ to distinguish source/target distributions, and then we update inference function to confuse $D$.
  5. The above process needs lots of unsupervised data.
  6. If there are only a few samples in $D_t$ but having labels, we can create pairs between them and samples in $D_s$. There are four groups, $G_1: x_s^{c_i}/x_s^{c_i}$, $G_2: x_s^{c_i}/x_t^{c_i}$, $G_3: x_s^{c_i}/x_s^{c_j}$, $G_4: x_s^{c_i}/x_t^{c_j}$.
  7. We want to create a discriminator $DCD$ to classify these 4 groups. Then we update $g_t$ to confuse $DCD$ between $G_1/G_2$ and $G_3/G_4$ as well as keep distinguishing between $G_1,G_2/G_3,G_4$.

Transferable Representations across Domains and Tasks

  1. We want to transfer information from a large labeled source domain $S$ to a sparsely labeled target domain $T$. $S$ and $T$ can be no overlapping.
  2. We assume the source domain contains $n^s$ labeled images. Target domain contains $n^t$ unlabeled and $m^t$ labeled images. $m^t \ll n^t$.
  3. We first train a model with $n^s$ and then copy the trained weights for target model.
  4. We build a multi-layer discriminator to align distribution of features from source and target domains(layers). For each discriminator, it takes samples and also the output from previous discriminator as joint inputs.
  5. We also define a new semantic transfer objective. For each unlabeled image, we calculate its similarity with all other labeled ones and then generate a softmax result. (Both $S$ and $T$). Besides, we calculate a metric based cross entropy loss. We calculate the similarity between each labeled sample in $T$ and all centroid of each class.
